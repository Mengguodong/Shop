中华人民共和国大陆地区退换货条款及细则
本使用条款及细则适用于买家在浏览及使用北京梵禹科技有限公司（以下简称为「本公司」）的电子商务平台－健康绿氧（以下简称为「本网站」），请于使用本网站前仔细阅读本条款及细则。买家使用本网站，即表示买家接受并同意受本条款及细则所约束。
1. 取消订单
1.1 买家提交订单后，即使在本公司仍未经电邮确认买家之订单之前，均不可取消订单；
1.1 买家可于商品签收后申请商品退货或换货，所有退货或换货申请必须符合本 条款内的商品退货或换货条件。
2. 商品退货或换货次数
2.1 每一商品可辨理退货或换货一次（只限于本公司认为符合退货或换货条件之商品）。
3. 接纳商品退货或换货之条件，包括但不限于：
3.1经本网站购买的商品；
3.2于7天有效商品退换期（由确认签收日起计7天内，包括周六、日及公休假）内提出的申请；
3.3商品外包装有损坏引致商品故障或破损；
3.4商品有瑕疵（如污迹、食品过期或不能正常使用等）或非人为因素造成之故障或破损；或
3.5送抵之商品与订购之商品不同。
4. 不接纳商品退货或换货之条件，包括但不限于：
4.1 任何非本网站出售的商品；
4.2 商品于本网站描述中注明不可退换；
4.3 于7天有效商品退货期后提出的申请（由确认签收日起计7天内，包括周六、日及公休假）；
4.4 商品并非全新状态，或因人为因素导致的损毁、商品无法回复原状、缺乏完整性及失去二次销售价值（如已穿着、使用、洗涤、受潮、湿损、自行剪裁修改、改装、污损、破损、凹陷、变形、功能不全、失灵等）；
4.5 礼品包或套装中的商品部分退换；
4.6 所有食品类；
4.7 贴身或卫生商品（适用于母婴产品类、宠物生活类、衣物服饰类、个人及家居护理类及运动用品类）；
4.8 包装已被拆封的商品（适用于母婴产品类、宠物生活类、衣物服饰类、游戏电玩类、个人及家居护理类、运动用品类、电子及电器产品类）；
4.9买家个人喜好问题；
4.10 非商品品质问题而提出的申请；
4.11 商品上的标签、条码或吊牌不完整、被破坏或被折曲；
4.12赠品（赠品于任何情况下均不获退换）；
4.13配件附属品瑕疵；
4.14 特卖商品；
4.15 商品所属商店另有列明退换货安排者；
4.16 外包装及商品无损坏；或
4.17 其他本公司认为不合理的退换货申请。
5. 申请退货或换货流程
5.1 如买家申请商品退货或换货，需先确定该商品属于可退货类型及符合商品退货或换货条件，并于商品确认签收日起计7天内（包括星期六、日及公众假期）登入本网站提出退货或换货申请；
5.2 为方便本公司进行买家退货或换货申请的审核，买家必需按要求将该商品照片连同退货理由一并附上。
6. 退货申请或换货审核
6.1 本公司收到买家的退货或换货申请后，会于3个工作天内审核买家的退货或换货申请，有关审核结果将以电邮通知买家；
6.2 如买家的退货或换货申请不获接纳，本公司将以电邮通知买家不予办理退货之原因。
7. 商品退货或换货地点
7.1 买家在本网站申请退货，经客户服务代表审核并接纳后，应于指定时间内自行到物流站将商品退回或安排收派员上门取回需退货的商品；
7.2 买家在本网站申请换货，经客户服务代表审核并接纳后，应于指定时间内自行到物流站将需更换的商品退回或安排收派员上门取回需更换的商品；
7.3 如买家欲了解物流站地址及营业时间，可参考第三方物流快递官方网站： http://www.sf-express.com/hk/sc/others/Contact_Us/SF_store_address/
8. 办理商品退货或换货
8.1 商品换货服务只限配送至中华人民共和国大陆地区收货地址之订单；
8.2如买家的商品退货或换货申请获本公司接纳，买家将收到电邮通知。本公司会于收到退回商品后安排退款或换货；
8.3任何退货或换货，买家须提交以下文件及物品：
8.3.1 接纳退货或换货申请之通知电邮列印本；
8.3.2 有关交易的完整订单列印本；及
8.3.3 需退货或换货商品（必须连同商品的完整包装）
8.4 以上的文件及物品如有遗漏，买家的退货或换货申请将不获办理；
8.5 如果买家选择上门进行退货或换货, 须于限定时间内将以上所需文件及商品一并退回，本公司确认收回需退货或换货的商品后，将安排退款或为更换的商品安排发货；
8.6 如果买家选择于物流站进行退货或换货, 须收到电话短讯提示后, 于限定时间内将以上所需文件及商品一并退回，本公司即场确认收回的商品后将安排退款或更换商品；
8.7 本公司安排的收派员会现场验收及核实退货或换货商品。如买家提交的商品与接纳退货或换货申请之通知电邮内容不一致，买家的退货或换货申请将不获办理；
8.8 本公司确认收回退货商品后将安排退款；
8.9 如买家未能于指定时间内将商品退回，买家的退货或换货申请将被视作放弃;
8.10 买家成功申请退货后，购买该商品所获得的积分将全数或按比例被扣除，而不作另行通知；
8.11 所有未经授权的退货或换货申请，本公司一律概不处理；
8.12如买家申请换货的商品缺货，本公司将依照退货申请处理，并安排退款；
8.13 退货或换货商品确认安排退回本公司仓库后，买家不得向本公司取回该商品。
